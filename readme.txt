== Description ==

UPDATE: Please use the http://wordpress.org/extend/plugins/wordpress-mobile-pack/ plugin instead, here's a brief description of the new plugin:

The dotMobi WordPress Mobile Pack is a complete toolkit to help mobilize your
WordPress site and blog.

It includes a mobile switcher to select themes based on the type of user that
is visiting the site, a selection of mobile themes, extra widgets, device
adaptation and a mobile administration panel to allow users to edit the site
or write new posts when out and about.

== Frequently Asked Questions ==

= Is Mowser still active? =

Mowser was sold to dotMobi, who have continued to provide tools and services
for publishers looking to reach users on mobile devices.  If you're looking
for a plugin to make your Wordpress site mobile check out their updated plugin:
http://wordpress.org/extend/plugins/wordpress-mobile-pack/
